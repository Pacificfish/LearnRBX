# Repeating Code with Loops

Learn to repeat code efficiently with for and while loops

## Learning Objectives

- Use for loops
- Use while loops
- Control loop execution

## Interactive Lesson

This lesson will guide you through hands-on coding exercises. Complete each step to progress through the lesson.


### Step 1: Explanation

Loops allow you to repeat code multiple times. For loops are great when you know how many times to repeat.
