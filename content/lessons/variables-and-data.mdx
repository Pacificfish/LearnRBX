# Variables and Data Types

Learn to store and work with different types of data

## Learning Objectives

- Create variables
- Understand data types
- Use local variables

## Interactive Lesson

This lesson will guide you through hands-on coding exercises. Complete each step to progress through the lesson.


### Step 1: Explanation

Variables are containers that store data. In Lua, we use the 'local' keyword to create variables.

---

### Step 2: Code Exercise

Create a variable called 'name' and store your name in it, then print it:

**Your Task:**
```lua
-- Create a variable called name
-- Store your name in it
-- Print the variable
```

**Expected Output:**
Complete the exercise above

**Hint:** Use 'local name = "Your Name"' to create the variable

---

### Step 3: Code Exercise

Now create a variable called 'age' and store a number in it:

**Your Task:**
```lua
-- Create a variable called age
-- Store a number in it
-- Print the variable
```

**Expected Output:**
Complete the exercise above

**Hint:** Numbers don't need quotes: local age = 25

---

### Step 4: Explanation

Perfect! You've learned about variables. In Lua, we have different data types: strings (text), numbers, booleans (true/false), and more.
