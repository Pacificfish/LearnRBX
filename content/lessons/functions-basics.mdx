# Creating Reusable Code with Functions

Organize your code with functions for reusability

## Learning Objectives

- Define functions
- Call functions
- Return values

## Interactive Lesson

This lesson will guide you through hands-on coding exercises. Complete each step to progress through the lesson.


### Step 1: Explanation

Functions are blocks of code that can be called multiple times. They help organize your code and make it reusable.

---

### Step 3: Code Exercise

**Your Task:**
